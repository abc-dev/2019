O ABC-Dev

O ABC-Dev é uma conferência anual realizada desde 2016 que atinge as pessoas desenvolvedoras de software da região do Grande ABC e São Paulo.

Este evento conta com o apoio e participação das comunidades locais FEMUG-ABC, CRUD-ABC, Front-Ladies ABC, GDG-ABC, ABC Hacker Club e com o trabalho brilhante de pessoas envolvidas nestas comunidades.

A conferência trata de assuntos importantes para as comunidades como: JavaScript, PHP, Java, .NET, Python, Ruby, Android, iOS, Testes de Software, DevOps, Cloud Computing, Big Data, Front-End, Arquitetura de Software, Agile e muito mais!

Contamos com o apoio e presença de importantes empresas de TI e comunidades do Brasil e do mundo.

Big "OpenSource" Data - Uma caixa de ferramentas abertas para grandes volumes de dados

Tiago de Freitas Lima

Vivemos na Era dos Dados, e extração de informações a partir de grandes quantidades de dados se tornou algo crucial. Esse contexto recebeu um nome: BigData, e as ferramentas de código aberto têm um papel-chave no desenvolvimento dessa tecnologia. Vamos passar pela história, pelas necessidades e desafios do BigData através de vários frameworks abertos, demonstrando como é possível construir uma arquitetura para processamento de grandes volumes de dados usando apenas ferramentas open source.

Machine Learning com Javascript utilizando Tensorflow.js

Carla Vieira

TensorFlow.js é uma biblioteca em Javascript para treinar e fazer deploy de modelos de Machine Learning, ela facilita: desenvolver modelos em Javascript, rodar modelos já existentes e retreinar modelos já existentes. Nesta palestra, irei apresentar as possibilidades de desenvolvimento utilizando o Tensorflow.js e, como exemplo, uma demo que utiliza um modelo de aprendizado de máquina do TensorFlow.js para identificação automática de comentários tóxicos.

O que não te contaram sobre microserviços

Breno Panzolini

Há algum tempo muito tem se falado sobre microserviços e suas grandes vantagens em relação à outras arquiteturas (principalmente aos monolítos). Porém, implementar uma arquitetura em microserviços vem acompanhada de muitos desafios e dificuldades que na maioria das vezes não são mostrados. Nessa palestra vamos tentar desvendar esses mistérios para que você realmente tenha noção do barco em que está entrando ao adotar uma arquitetura de microserviços.
